    Console

    console.log()

    console.log() takes multiple arguments
    Label your debug output! 

    
console.log("Hello World");
console.log(1 + 2);

var foo = "bar";
console.log(foo);
console.log("foo:", foo);
foo = true;
console.log(!foo);
//also handy!
console.info('info:', foo);
console.warn('warn:', foo);
console.error('error:', foo);

    console.group

    console.group(object[, object, ...])

    Starts a new logging group with an optional title. All console output that occurs after calling this method and calling console.groupEnd() appears in the same visual group.

    
console.group("Authenticating user '%s'", user);
console.log("User authenticated");
someOtherLongFunction();
console.groupEnd();

    Debugging With debugger;

    Debugger

    You can automatically create a breakpoint in your code by adding the following line to your JavaScript: debugger; . This will automatically create a breakpoint in the code when it is run with Developer Tools open.
